
# Controls

## Normal Mode:
	Go Forward   (Long Press)
	Look Around  (Pan)
	Reset Camera Position  (Double Tap)
	Open Mini-map (Two Fingers Double Tap)

	Enter Model Edit (Three Fingers Single Tap)

## In Edit Mode:
	Exit Model Edit (Three Fingers Single Tap)
	Move Enemy Model (One Finger Pan)
	Rotate Enemy Model (Two Finger Pan)
	Rescale (Pinch)

# Buttons: 
Switch 1: Day/Night mode
Switch 2: Turn On/Off flash light
Switch 3: Turn On/Off fog
Fog Intensity: Change Fog Intensity when fog is on
